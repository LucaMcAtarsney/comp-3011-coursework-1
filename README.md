# Player Stats API

This project is a FastAPI-based backend for tracking player statistics and game runs for a video game.

## Features

- Player management (create, list)
- Game run tracking (start, update, end)
- Leaderboards
- Player analytics and statistics

## Tech Stack

- **Backend:** FastAPI
- **Database:** SQLAlchemy with SQLite (default)
- **Data Validation:** Pydantic

## Getting Started

### Prerequisites

- Python 3.8+
- Poetry (or pip) for dependency management

### Installation

1.  Clone the repository:
    ```sh
    git clone <your-repo-url>
    cd web-course-work-1
    ```

2.  Install dependencies:
    ```sh
    pip install -r requirements.txt
    ```

3.  (Optional) For development, install testing libraries:
    ```sh
    pip install -r requirements-dev.txt
    ```

### Running the Application

To run the development server:

```sh
uvicorn main:app --reload
```

The API will be available at `http://127.0.0.1:8000`.

### Running Tests

To run the automated tests:

```sh
pytest
```

## API Endpoints

The API documentation is automatically generated by FastAPI and can be accessed at:

-   **/docs**: Interactive Swagger UI
-   **/redoc**: ReDoc documentation

### Authentication

Most endpoints are protected by an API key. Include the key in the `X-API-KEY` header with your requests.
