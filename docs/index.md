# Welcome to the Player and Run Tracker API

This documentation provides a complete reference for the API, including all available endpoints, request parameters, and response schemas.

## Overview

The API is built with FastAPI and provides endpoints for:

- Player management (creation, authentication, stats)
- Game run tracking (start, update, end)
- Analytics (leaderboards, player summaries)
- Admin operations

Navigate to the **API Reference** section to see the detailed endpoint documentation.
